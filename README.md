# Balloon Party

A cute and simple web project where you can release colorful balloons using only the **B**.

## About

Every time you press the **B**, a new balloon appears from the bottom of the screen and gently floats into the sky.

Each balloon has:
- A random pastel color
- A random size
- Play a Woosh sound
A counter at the top keeps track of how many balloons you've released.
after every 10 count of balloons there will be  Confetti celebration

## Tech Stack
- HTML
- CSS
- JavaScript

## Why I Made This
The goal was to practice:
- Creating HTML elements with JavaScript
- CSS animations
- Random colors and size
